Tags Mais Comuns no HTML

O que são as Tags no HTML

Em HTML, as tags são elementos usados para marcar e estruturar o conteúdo em uma página da web. Elas são cercadas por colchetes angulares "< >" e podem ter atributos para fornecer informações adicionais. Existem tags de abertura e tags de fechamento, sendo que as últimas têm uma barra antes do nome, como <tag> e </tag>, respectivamente. Tags de abertura são usadas para iniciar um elemento, enquanto as de fechamento são usadas para encerrá-lo. Exemplos comuns de tags incluem <p> para parágrafos, <a> para links e <img> para imagens. Juntas, as tags HTML definem a estrutura e o conteúdo de uma página da web.

<h1 a h6>

(voltar)

As tags <h1>, <h2>, <h3>, <h4>, <h5>, e <h6> são usadas para criar títulos e subtítulos em uma página HTML, com <h1> sendo o título mais importante e <h6> o menos importante. Elas são usadas para estruturar o conteúdo da página e ajudar na organização e acessibilidade.

a escolha adequada do nível de título é importante para a estrutura e a semântica do seu conteúdo HTML, bem como para a acessibilidade da página. Geralmente, <h1> é usado para o título principal da página, e os níveis de título diminuem à medida que você cria subdivisões de conteúdo.

<hr>

(voltar)

A tag <hr> em HTML é usada para criar uma linha horizontal (ou "horizontal rule"). É usada para criar uma separação visual entre seções de conteúdo em uma página. Trata-se de uma tag vazia, o que significa que não possui um elemento de fechamento correspondente.

<p>

(voltar)

A tag <p>> em HTML define um parágrafo de texto em uma página da web. É uma das tags mais básicas e amplamente usada para separar blocos de texto e criar uma formatação visual de parágrafos na página.

<mark>

(voltar)

A tag <mark> em HTML é usada para destacar parte do texto dentro de um elemento. Ela é usada para aplicar destaque visual ao texto, geralmente com uma cor de fundo amarela por padrão, mas a aparência pode ser personalizada com CSS.

<small>

(voltar)

A tag <small> em HTML é usada para indicar que o texto dentro dela deve ser exibido em um tamanho de fonte menor do que o texto circundante, geralmente para indicar texto de menor importância ou subtexto.

<i, u e strong>

(voltar)

A tag <i> em HTML é usada para definir texto como itálico, mas não implica ênfase semântica. É geralmente usado para destacar texto com estilo itálico por razões de formatação, não de significado.

A tag <u> em HTML é usada para sublinhar texto. No entanto, não é recomendada para fins semânticos, pois sublinhar geralmente é associado a links. Use CSS para estilizar o texto com sublinhado.

A tag <strong> em HTML é usada para indicar ênfase forte no texto. Geralmente, o texto dentro desta tag é exibido em negrito, e a ênfase é semântica, indicando que o texto é importante.

<ol, ul e li>

(voltar)

A tag <ol> em HTML é usada para criar listas ordenadas, onde os itens são numerados ou classificados de alguma forma. Cada item da lista é definido com a tag <li>.

A tag <ul> em HTML é usada para criar listas não ordenadas, onde os itens não são numerados ou classificados. Cada item da lista é definido com a tag <li>.

A tag <li> em HTML é usada dentro de elementos <ol> (lista ordenada) ou <ul> (lista não ordenada) para definir cada item da lista.

Essas tags são amplamente usadas para criar listas em páginas da web e ajudam a organizar e estruturar informações de maneira clara e legível.

A seguir um exemplo de lista ordenada e não ordenada, respectivamente

    Lista Ordenada
  1. Item 1
  2. Item 2
  3. Item 3
    Lista NÃO Ordenada
  • Item 1
  • Item 2
  • Item 3

<a>

(voltar)

A tag <a> em HTML é usada para criar hiperlinks em HTML. Ela permite que você vincule o texto ou outros elementos a outras páginas da web, arquivos, endereços de email, ou âncoras dentro da mesma página. O atributo href especifica o destino do link.

<sub e sup>

(voltar)

A tag <sub> em HTML é usada para criar texto subscrito, que é posicionado ligeiramente abaixo da linha de base. É comumente usado para representar fórmulas químicas, notas de rodapé ou outras informações que devem aparecer em uma posição inferior.

A tag <sup> em HTML é usada para criar texto sobrescrito, que é posicionado ligeiramente acima da linha de base. É frequentemente usado para indicar expoentes, como em fórmulas matemáticas ou datas ordinais.

Essas tags são usadas para ajustar a posição vertical do texto em relação à linha de base, permitindo que você represente informações de forma mais precisa e legível.

<blockquote>

(voltar)

A tag <blockquote> em HTML é usada para criar um bloco de citação em HTML. Geralmente, o texto dentro desta tag é recuado para indicar que é uma citação de outra fonte.

<font>

(voltar)

A tag <font> em HTML era usada em versões mais antigas do HTML para definir o estilo da fonte, como tamanho, cor e família de fonte. No entanto, ela foi descontinuada no HTML5 em favor do uso de CSS para estilizar elementos HTML. Portanto, a tag <font> não é recomendada para uso em documentos HTML modernos.

Em vez disso, você deve usar CSS para estilizar seu texto em HTML. Isto é uma abordagem moderna e semântica para estilizar o texto em HTML em conformidade com os padrões atuais.

<del>

(voltar)

A tag <del> em HTML é usada para marcar o texto que foi deletado ou removido de uma versão anterior do conteúdo. Geralmente, o texto dentro desta tag é exibido com uma linha através dele para indicar que foi eliminado.

Exemplo de texto marcado como removido (deletado):

Este texto foi removido!
